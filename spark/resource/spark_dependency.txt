Build all necessary jars
========================
in chombo
mvn clean install
sbt publishLocal

in chombo/spark
sbt clean package
sbt publishLocal

in hoidla
mvn clean install
sbt publishLocal

in ruscello/spark
sbt clean package

Build uber jar
==============
ant -f ruscello.xml

uber jar file name is uber-ruscello-spark-1.0.jar

If you are using Spark 2.0+, please add the following line to ruscello.xml, because 
type safe jar is not included in newer versions of Spark

<zipgroupfileset dir="/Users/pranab/.ivy2/cache/com.typesafe/config/jars" includes="config-1.2.1.jar" />

Please change the directory path, as per your environment
